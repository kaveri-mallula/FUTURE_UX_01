# 💇‍♀️ FUTURE_UX_01 – Glow Salon Website Redesign

## 📌 Task Objective
To redesign a local salon website focusing on user experience and conversion optimization.

## 🎯 Problem
Many salon websites lack clear service structure and easy booking flow, leading to poor user experience.

## 💡 Solution
Designed a clean and modern UI that simplifies navigation and improves booking experience.

## 🚀 Key Features
- Clean homepage with service highlights
- Organized service listing
- Simple booking/contact section
- Responsive design

## 🧠 UX Decisions
- Focused on minimal and clean layout
- Easy navigation for users
- Highlighted important services

## 🛠 Tech Stack
- React
- Vite
- UI Components
